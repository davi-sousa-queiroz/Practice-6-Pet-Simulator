# 🐶 Pet Simulator (CLI)

A simple interactive pet simulator I built while learning Python classes.

This project might look small, but it was a big step for me. It’s where classes actually started to click.


## 🎮 What it does

You can:

* Feed your pet 🍖
* Play with your pet 🎾
* Check its stats 📊
* Watch it change over time

All through a simple terminal menu.

## 🧠 What I learned

This project helped me understand:

* How to use **classes** properly
* Why `self` exists (this confused me at first 😭)
* How to connect **objects + methods + user input**
* How to build a simple interactive loop

Big realization:

> Classes aren’t just syntax, they make managing state WAY easier.

---

## 😵 What confused me

* I kept trying to use variables like `my_pet` *inside* the class instead of `self`
* I forgot to pass parameters into the class at one point and nothing worked 💀
* Calling methods inside a loop felt weird at first

---

## 💡 What I liked

* Seeing the pet actually change over time felt like a real system
* The menu + input made it feel like a mini game
* Way more fun than just running functions manually
